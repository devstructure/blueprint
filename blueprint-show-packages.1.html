NAME

blueprint-show-packages - show packages in a blueprint

SYNOPSIS

blueprint show-packages [-q] [name]

DESCRIPTION

blueprint-show-packages prints the manager, name, and version (separated by spaces) of each package in the blueprint name. They are printed one-per-line to make this command useful in pipelines.

If name is omitted or -, a blueprint is read from standard input and treated in the same manner. See blueprint(5) for the details of the format.

OPTIONS

-q, --quiet
Operate quietly.
-h, --help
Show a help message.

FILES

~/.blueprints.git
The local repsitory where blueprints are stored, each on its own branch.
/etc/blueprintignore, ~/.blueprintignore
Lists of filename patterns to be ignored when creating blueprints. See blueprintignore(5).
